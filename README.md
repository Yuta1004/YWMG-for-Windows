# YMWG for Windows

## YMWGとは
弓削商船Webメールを自動で取得し、新着があれば表示します  

## インストール方法
**install.py**を実行して指示に従って必要情報を入力して下さい。  

## 注意
初期起動時はメールファイルが空になっているため、最新15件までを新着メールとして表示します。  
次回起動時からは新着メールのみを表示するようになります。  
新着メールがある場合はウィンドウが画面に残り続けますが、新着メールがない場合は自動的にウィンドウが閉じられます。
